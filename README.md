# Localization Skills for Claude

AI 深度本地化提示词集合，用于将英文 SaaS 营销博客文章深度改写（非翻译）为目标语言的地道内容。

## 支持语言

| 文件 | 语言 | 说明 |
|------|------|------|
| `localization-zh.md` | 简体中文 | 公众号风格、短句驱动、段落重组 |
| `localization-ja.md` | 日语 | ですます体、起承転結、主语省略 |
| `localization-ko.md` | 韩语 | 해요体、超短段落、共感型吐槽 |
| `localization-es.md` | 西班牙语 | tú 体、拉美优先、叙事性强 |
| `localization-de.md` | 德语 | du 体、Sachlichkeit、反美式 hype |
| `localization-fr.md` | 法语 | vous 体、论证严密、文风优雅 |

## 提示词结构

每份提示词包含以下维度：

1. **角色定义** — 用目标语言写，帮 AI "入戏"
2. **文化场景替换** — 西方隐喻→目标文化等价表达
3. **句式结构** — 目标语言特有的句式规则
4. **语气与风格** — 参考媒体 + 幽默方式
5. **标题与小标题** — 改写规则
6. **篇章结构重组** — 段落/章节/论证逻辑调整
7. **排版规范** — 标点、数字、外来语等
8. **结构完整性保护** — 链接/图片/组件不可修改
9. **禁止表达** — 翻译腔清单 + 替代方案
10. **深层思维差异** — 英语与目标语言的写作思维差异
11. **工作流程** — 三步流程：计划→确认→改写

## 使用方式

1. 将对应语言的提示词作为 system prompt 或对话开头发送给 Claude
2. 粘贴英文原文（Markdown 格式）
3. AI 会先输出重写计划，等你确认后再执行改写

## 中文注释

非中文提示词中，所有目标语言描述后都有 `> 💬` 引用块形式的中文翻译，方便中文团队理解。
